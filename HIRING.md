# System Design Assistant — HIRING

Minimal dedicated hiring; reuses portfolio team + engines (#2/#1). Portfolio principles: BUSINESS_STRATEGY.md §5.

- **Stage 0 (solo):** founder builds MVP (~6 wks) on the shared stack, reusing #2 grounding + #1 memory + `packages/evals`.
- **Stage 1 (shared):** portfolio full-stack engineer maintains it; no separate org.
- **Content/SEO help:** interview-prep SEO benefits from a content/DevRel contributor (portfolio DevRel or a contractor writer) — high-leverage for the prep funnel.
- **Later:** if interview-prep grows materially, a part-time content lead; otherwise stays a lean module.

**Principle:** don't staff a tertiary product; let it ride the shared team/engines. Interview-prep's main "hire" is content firepower, not engineering.
