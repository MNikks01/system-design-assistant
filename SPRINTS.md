~4–6 weeks to MVP. Build after the wedge (reuses #2 for grounding, #1 for memory).

- **S1 (Wk1–2):** Design conversation (Opus) + Mermaid generation + validation + diagram canvas. **Exit:** describe a problem → get a valid architecture diagram + options.
